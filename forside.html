Webserver

En webserver er en programvare som lagrer og sender ut data. En webserver skal behandle HTTP forespørsler fra en webbrowser, som for eksempel Opera, Chrome og Internet Explorer.

Domener

URL er en standard for angivelse av adresser til dokumenter, bilder og andre filer lokalt på datamaskinen eller på internett. URL blir ofte kalt nettadresse.

Les mer om Domener

MathML

MathML er måte å utrykke en matematisk formel på i HTML. MathML står for Mathematical Markup Language.

Les mer om MathML

SVG

SVG er en måte å tegne på. SVG er nesten akkurat det samme som å tegne i et canvas.

Les mer om SVG

Git og Git-hub

Git er et verktøy for visjonskontroll og kodedeling. Git-hub er et verktøy som gjør det enkelt å sammarbeide med andre.
